MZH010开发板使用记录

发表于 2021-07-08 更新于 2021-07-09 分类于 硬件学习

记录MZH010开发板的使用方法等

01 Keil uVision的安装使用

源文件见E:\study\SoftwareUserGuides\Keil_uVision。

照抄正点原子的教程：

请按如下顺序安装：

1，安装mdk_514.exe
2，安装Keil.STM32F1xx_DFP.1.0.5.pack
3，注册方法：用注册机生成序列号

注册机可以用到2032年，根据Keil官方说法，快过期的时候就会发布截止日期更晚的注册机。

02 JLINK安装

双击E:\study\SoftwareUserGuides\ST-LINK-DRIVER\ST-LINKV2 USB driver for Windows 8 10\ST-LINKV2 USB driver for Windows 8\stsw-link006\stlink_winusb_install.bat安装即可，然后就可以在Keil里直接烧录程序了。

03 阿里云平台

用户名：nju_test_iot

控制台->物联网平台->公共实例->设备

实录

  1. 需要micro USB线一根和JLINK一根。
  2. MZH010的资料：E:\ICAIS\projects\insulator\MCU
